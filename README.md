# 
#### **TestTaskAnsoft**

Приложение состоит из 3 экранов:
 - Список доставок для выполнения
 - Список выполненных доставок
 - Карта доставок - отражаются все доставки

#### Стек технологий
- MVVM
- Kotlin Coroutines
- Flow
- Retroffit 2
- Room
- Navigation component
- Koin
